# Documentation of DRENGRFELL

## Keybinds

| Key      | Function                       |
|----------|--------------------------------|
| F        | Toggle rendering of fog of war |
| G        | Regenerate map                 |
| W A S D  | Move map                       |
| N        | Preview settlements            |
| B        | Preview roads                  |
| + -      | Zoom                           |
